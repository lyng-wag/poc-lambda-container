# poc-lambda-container

Repo for testing out https://docs.aws.amazon.com/lambda/latest/dg/lambda-images.html

## Instructions

1. Build: `./build.sh`

2. Start Container: `./startLocal.sh`

3. Trigger Lambda in Container: `./callLambda.sh`
